HTML학습하기

H태그는 제목을 의미하고 위아래로 약간의 여백이 자동삽입되고 자동줄바꿈 됨

검색엔진에서 h태그는 중요한 검색 키워드로 사용

p태그는 단락구분태그이고, 줄바꿈이 되면서 태그 위아래로 약간의 여백이 삽입

html 태그 내에서 엔터를 치더라도 줄바꿈이 되지 않고,
또한 웹브라우저에서는 "<br>" 을 넣어야함 또한, 웹브라우저에서는 여러번 띄어쓰기를 하더라도 한 번만 띄어쓰기가 됨에 유의.

텍스트 형식

굵게와 강조된 굵게가 사용되고, 이텔릭체와 강조된 이텔릭체가 있다. 삭제된 텍스트를 표현할 때 del을 사용한다.

span태그는 별 다른 기능이 없고, html 요소들에 css 적용을 위해 주로 사용. 줄바꿈이 없다.
